# Mintlify AI Toolkit - Web Scraper

Browser-based documentation scraper for Mintlify sites.

## Features
- Client-side scraping (no server needed)
- Automatic Trieve API detection
- ZIP download of complete documentation
- Clean markdown output

## Usage
Visit https://nirholas.github.io/mintlify-ai-toolkit to use the web interface.

## Files
- `index.html` - Main web interface
- `app.js` - Scraping logic
- Dependencies loaded from CDN (JSZip)

## Local Development
Simply open `index.html` in a browser. No build process required.

## Deployment
Automatically deployed to GitHub Pages via GitHub Actions on push to main branch.
